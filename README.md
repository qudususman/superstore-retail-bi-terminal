# Enterprise 5-Page Supply Chain & Financial Intelligence Terminal

## 🚀 Live Portfolio Project Overview
An executive-ready, high-density Power BI business intelligence application engineered to transform raw, unstructured transactional data into an interactive visual cockpit. The entire front-end architecture is wrapped in a premium, ultra-modern **Deep Charcoal & Matte Velvet Theme (#0F0F12)** with strict metric alignment to minimize data cognitive load for C-suite decision-makers.

## 🛠️ Technical Data Framework Deployed
*   **Power Query ETL Ingestion:** Advanced data extraction, row filtering, date skyrocketing, and data cleansing to eliminate duplicate anomalies.
*   **Relational Star Schema Modeling:** Strict architectural separation between transactional Fact tables and lookup Dimension tables to optimize query performance and protect cross-filtering context.
*   **Advanced DAX Logic Layer:** Development of custom calculation measures from scratch tracking dynamic running totals, Year-over-Year (YoY) growth, and true Profit Margin % metrics.
*   **C-Suite Interaction Suite:** Structured navigation tabs separating metrics into private analytical rooms (Executive Overview, Customer Profiles, Product Performance, and Regional Matrices).

## 🕵️‍♂️ Prescriptive Analytical Case Insights (Management by Exception)
*   **The Champion Benchmark:** Isolated the **West Region** as the primary profit powerhouse ($108.42K Net Profit, 14.94% Margin) driven by solid Technology product velocity.
*   **The Margin Leakage Exposure:** Detected a severe margin crisis inside the **Central Region** ($39.71K Profit, 7.92% Margin), uncovering a system-wide discount vulnerability where bulk consumer accounts (e.g., Sean Miller) generated deep net-loss deficits despite high revenue volume.
*   **The Inventory Optimization:** Exposed severe freight cost erosion across **Furniture lines (Tables/Bookcases)**, providing the corporate strategy matrix to deploy hard-cap checkout discount guardrails and enforce a customer-paid shipping infrastructure to insulate bottom-line margins.
